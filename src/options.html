APIキーの取得方法:
  1. Google Cloud Console でプロジェクトを作成
  2. YouTube Data API v3 を有効化
  3. 認証情報でAPIキーを作成
  4. キーの制限で「HTTPリファラー」を選択し、「https://www.youtube.com/*」を追加
